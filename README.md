#  Spam Email Classifier

##  Project Overview

This project uses Machine Learning and Natural Language Processing (NLP) to classify messages as **Spam** or **Not Spam**.

---

##  Features

* Text preprocessing (cleaning, stopword removal)
* TF-IDF Vectorization
* Model training using Naive Bayes
* Prediction on new messages

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

---

##  Dataset

* File: `spam.csv`
* Contains labeled SMS messages (spam/ham)

---

##  Steps Involved

1. Data Loading
2. Data Cleaning
3. Feature Extraction (TF-IDF)
4. Model Training
5. Model Evaluation
6. Prediction
